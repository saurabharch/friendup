Introduction

The Friend Create IDE is meant to be a simple but powerful tool to create Friend applications (and server modules). It is tuned to help you work on code, single and multiple projects as well as packages for distribution.

Friend Create supports the functionality you would come to expect from an entry level programmers editor, but tries to get our of your way and let you do most of your work.

Friend packages

When building a Friend package, you need to first create a project. The project keeps a structure of your code and assets so that is can be converted into a package for distribution.

Before you can build a package, you need to make sure you have the following structure available:

  • Project name - a unique name of your project
  • Project author - who created the app
  • Project category - from a pulldown list
  • Project version - to enable upgrade controls
  • Project description - describing your application in the marketplace
  • screenshot.jpg - the image that shows your app in the marketplace (optional)
  • preview.jpg - the preview image for the marketplace
  • icon.png - the icon for your application

If these criteria are met, you will be able to build a Friend package and install it on your Friend server - and you will be able to submit it to Friend Software Corp. for review and distribution on Friend Sky.

To execute the building of your package, open your project editor on your active project and fill out the form. Remember to check that all of your files are represented in the images and files list. Then, you can use the Packages -> Generate package menu item in the main Friend Create application menu. The package will be generated and placed in your project folder.
