Friend Create er et integrert utviklermiljø for Friend OS. Med dette programmet kan du utvikle alt fra Friend applikasjoner, widgets, kommandolinje programmer og server programvare i flere språk.

Friend Create var ett av de første programmene som ble utviklet med bruk av Friend's forskjellige APIer og biblotek. I versjon 2.0 er hele programmet refaktorert, og et helt nytt brukergrensesnitt er utviklet.

Friend Create er Friend Software's tilbud til utviklere som trenger et enkelt og effektivt program for å utvikle programvare. Det er designet for å gjøre det enkelt å lage pakker for distribusjon i Friend Network og Friend Marketplace.

Forfatteren

Friend Create er utviklet av Hogne Titlestad og ble først satt sammen i 2014.
